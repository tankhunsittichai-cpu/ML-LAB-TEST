## LAB2: Data Visualization

This repository section focuses on **Data Visualization**, which involves translating data into a graphical context. Visualizing the data makes it easier to identify trends, patterns, and relationships within the dataset.

### Key Visualizations Performed:

*   **Histogram:** Used to visualize the distribution of continuous numerical variables. It helps in understanding the frequency, spread, skewness, and potential outliers within individual features.
*   **Correlation Heatmap:** A graphical representation of a correlation matrix that shows the strength and direction of relationships between multiple numerical variables. It helps in identifying highly correlated features, which is essential for feature selection.
