## LAB1: Dataset Exploration

This repository contains the first laboratory assignment focused on **Dataset Exploration (EDA - Exploratory Data Analysis)**. The goal of this lab is to understand the structure, characteristics, and quality of the dataset before applying any machine learning algorithms.

### Key Steps Performed:

*   **Load Dataset:** Importing the dataset into the working environment (e.g., using Pandas in Python) to prepare it for analysis.
*   **Display Shape:** Checking the dimensions of the dataset to know the total number of rows (samples) and columns (features).
*   **Display Data Types:** Inspecting the data type of each column (e.g., numerical, categorical, text) to ensure they are correctly interpreted.
*   **Display Summary Statistics:** Generating statistical summaries (such as mean, median, min, max, and standard deviation) to understand the distribution of numerical data.
*   **Display Missing Values:** Identifying any missing or null values across the dataset that need to be handled during data cleaning.
*   **Display Duplicate Records:** Checking for and identifying redundant or duplicated rows in the dataset.
*   **Display Class Distribution:** Analyzing the distribution of the target variable/class labels to check if the dataset is balanced or imbalanced.
